# Notebooks

Esta carpeta contiene los notebooks del portfolio, organizados por proyecto.

## Organización

Cada subcarpeta corresponde a un notebook o proyecto independiente. El prefijo numérico indica el orden de publicación, no necesariamente el orden cronológico de realización.

| `01_energy_economics_clustering/` | Energy Economics Clustering | Pendiente — notebook por añadir |

| `03_retinal_fundus_image_classification/` | Retinal Fundus Image Classification | Pendiente — notebook por añadir |

## Convenciones de nombres

- Las carpetas usan prefijo numérico y guiones bajos: `NN_nombre_descriptivo/`
- Los notebooks publicados se renombran con un nombre descriptivo en inglés: `nombre_descriptivo.ipynb`
- Los archivos originales del curso (con nombres de entrega) **no se publican directamente**. Se limpian y renombran antes de añadirlos aquí.

## Proceso antes de publicar un notebook

1. Limpiar outputs innecesarios o muy grandes.
2. Revisar que no haya rutas locales, datos privados ni enunciados del curso.
3. Renombrar el archivo con un nombre profesional y descriptivo.
4. Actualizar el README de la carpeta correspondiente.
5. Actualizar el badge "Open in Colab" en el README principal.
6. Revisar el checklist en [`../docs/PUBLISHING_CHECKLIST.md`](../docs/PUBLISHING_CHECKLIST.md).
