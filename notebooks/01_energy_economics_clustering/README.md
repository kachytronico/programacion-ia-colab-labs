# 01 — Energy Economics Clustering

## Objetivo

Explorar patrones en datos económico-energéticos mediante técnicas de clustering no supervisado, con posterior interpretación mediante un modelo supervisado.

## Notebook original

Archivo original (local, no publicado): `Ledesma_Ruiz_Alfredo_PIA_P1.ipynb`

El archivo publicado en este repositorio se llamará: `energy_economics_clustering.ipynb`

## Técnicas principales

- Preprocesamiento de datos tabulares.
- Validación temporal de los datos.
- Análisis de correlación.
- Reducción de dimensionalidad con PCA.
- Clustering con KMeans y métodos aglomerativos.
- Pseudo-etiquetado de clusters.
- Interpretación con modelo supervisado.
