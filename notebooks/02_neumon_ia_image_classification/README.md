# 02 — Neumon-IA Image Classification

## Objetivo

Desarrollar un flujo de trabajo completo de clasificación de imágenes para el diagnóstico diferencial de neumonía, evaluando distintas arquitecturas y analizando errores críticos.

## Notebook original

Archivo original (local, no publicado): `Ledesma_Ruiz_Alfredo_PIA_P2.ipynb`

El archivo publicado en este repositorio se llamará: `neumon_ia_image_classification.ipynb`

## Técnicas principales

- Inspección y limpieza del dataset de imágenes.
- División estratificada del dataset.
- DataBlock y pipeline de datos con fastai.
- Modelo baseline de clasificación profunda.
- Optimización de hiperparámetros.
- Comparación de arquitecturas.
- Matriz de confusión y análisis de errores críticos.
