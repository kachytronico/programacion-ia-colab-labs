# 03 — Retinal Fundus Image Classification

## Objetivo

Practicar un flujo de trabajo de visión por computador con imágenes de fondo de retina, poniendo énfasis en la métrica personalizada para minimizar falsos positivos y en el análisis crítico de resultados.

## Notebook original

Archivo original (local, no publicado): `Simulacro de proyecto 2024 P2.ipynb`

El archivo publicado en este repositorio se llamará: `retinal_fundus_image_classification.ipynb`

## Técnicas principales

- Modelo baseline de clasificación de imágenes.
- Búsqueda de learning rate óptimo.
- Early stopping.
- Data augmentation.
- Matriz de confusión.
- Métrica personalizada centrada en falsos positivos.
